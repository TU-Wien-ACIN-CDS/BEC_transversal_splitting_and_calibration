Code for results described in https://arxiv.org/abs/2510.07113

Featuring 4 destinct features:

1. Generating a set of identification experiments either through a heuristic method or optimization using a genetic algorithm
   The genetic algorithm calculates the Sensitivities if the parameters are used for the first time, can take a long time.
	- main_experiment_selection.m

2. Identifying the parameters for a given set of identification experiments and mesured data
	- main_calibrate_model_from_measurements.m

3. Optimizing trajectories using the identified parameters
	- main_optimize_trajectories.m

4. Simulating the results of the optimized trajectories
	- main_simulate_trajectory_results.m



Folder Structure
- functions 	 	holds the internal functions
- Parameters 		holds the constant parameter file
- Storage 		holds all generated data
  - Calibrated_parameters 	holds the parameters generated by the calibration, allready includes best january and march calibrations
  - Calibration_Szenarios 	holds the calibration szenarios
  - Optimized_Ramps 		holds the optimized ramps
  - Sensitivities 		holds the sensitivities for the genetic algorithm
  - Simulated_Data 		holds the data simulated from trajectories
  - Simulated_Initial_States 	holds the data of simulated initial states used for further optimization



MATLAB Packages required:
- Optimization Toolbox
- Signal Processing Toolbox
- Symbolic Math Toolbox
- Image Processing Toolbox
- Curve Fitting Toolbox
